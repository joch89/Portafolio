Proyectos de Machine Learning

🧠 Predicción de Cáncer de Mama (Proyecto MLOps)

Descripción: Desarrollo de un pipeline completo de Machine Learning (MLOps) para la clasificación de tumores de mama (benignos/malignos) basado en el dataset de Wisconsin. Este proyecto se enfoca no solo en la precisión del modelo, sino también en la automatización y puesta en producción.

Objetivos:

  • Implementar un modelo de clasificación con alta sensibilidad y especificidad.
  • Configurar un flujo de trabajo MLOps (versionamiento de datos, entrenamiento, registro y despliegue).
  • Demostrar la capacidad de llevar un modelo de la experimentación a la producción.

Tecnologías utilizadas: Python, Scikit-learn, MLflow, DVC, Docker, GitHub Actions, Streamlit.

Reflexión: "Este proyecto fue fundamental para comprender que la Ciencia de Datos en un entorno real no termina con el entrenamiento del modelo. La implementación de MLOps con MLflow y DVC me permitió aprender sobre la trazabilidad y la reproducibilidad, aspectos críticos en aplicaciones sensibles como la salud."
